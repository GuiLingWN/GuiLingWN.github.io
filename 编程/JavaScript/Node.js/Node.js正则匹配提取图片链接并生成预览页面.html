# Node.js正则匹配提取图片链接并生成预览页面

在处理 HTML 或 JavaScript 文件时，有时需要提取其中所有的图片链接并生成一个预览页面。这里分享一个使用正则表达式批量提取图片链接的 Node.js 脚本。

# 使用场景

  • 从编译后的前端项目文件中提取所有图片资源
  • 快速预览项目中使用的所有图片
  • 检查图片链接是否正确
  • 批量处理图片资源

# 脚本实现

/**
 * 匹配文件中所有图片超链接并输出到1.html中显示
 */
const fs = require('fs')
const filePath = `${__dirname}/page-frame.html`
// const filePath = `${__dirname}/1.js`

let content = fs.readFileSync(filePath, 'utf-8')
let srcReg = /'|\\")https:\/\/([^\\'\\"]*)\\.(jpg|png|gif)\\1/gi;
let arr = content.match(srcReg);

console.log('所有已成功匹配图片的数组：' + arr);

let html = ''
arr.forEach(url => {
  html += `<img style=\"border:1px solid red;\" src=${url}/>\n`
})

fs.writeFileSync(`${__dirname}/1.html`, html)

# 代码说明

# 1. 读取源文件

let content = fs.readFileSync(filePath, 'utf-8')

读取待处理的 HTML 或 JS 文件内容。

# 2. 正则匹配图片链接

let srcReg = /'|\\")https:\/\/([^\\'\\"]*)\\.(jpg|png|gif)\\1/gi;
let arr = content.match(srcReg);

正则表达式说明：

  • 匹配以 https:// 开头的图片链接
  • 支持 jpg、png、gif 三种图片格式
  • 使用全局匹配（g 标志）和忽略大小写（i 标志）

# 3. 生成 HTML 预览页面

let html = ''
arr.forEach(url => {
  html += `<img style=\"border:1px solid red;\" src=${url}/>\n`
})
fs.writeFileSync(`${__dirname}/1.html`, html)

将所有图片链接生成 <img> 标签，并添加红色边框便于查看。

# 改进建议

# 1. 支持更多图片格式

let srcReg = /'|\\")https:\/\/([^\\'\\"]*)\\.(jpg|jpeg|png|gif|webp|svg)\\1/gi;

# 2. 生成完整的 HTML 页面

let html = `
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>图片预览</title>
  <style>
    img {
      margin: 10px;
      border: 1px solid #ccc;
      max-width: 300px;
    }
  </style>
</head>
<body>
  <h1>共找到 ${arr.length} 张图片</h1>
`
arr.forEach((url, index) => {
  html += `  <div>
    <p>${index + 1}. ${url}</p>
    <img src="${url}" />
  </div>\n`
})
html += `
</body>
</html>
`
fs.writeFileSync(`${__dirname}/preview.html`, html)

# 3. 添加错误处理

try {
  let content = fs.readFileSync(filePath, 'utf-8')
  let srcReg = /'|\\")https:\/\/([^\\'\\"]*)\\.(jpg|png|gif)\\1/gi;
  let arr = content.match(srcReg);
  
  if (!arr || arr.length === 0) {
    console.log('未找到任何图片链接')
    return
  }
  
  console.log(`成功匹配 ${arr.length} 张图片`)
  // 生成 HTML...
} catch (error) {
  console.error('处理失败:', error.message)
}

# 使用方法

  1. 将代码保存为 extract-images.js
  2. 修改 filePath 为你的目标文件路径
  3. 运行脚本：node extract-images.js
  4. 在浏览器中打开生成的 1.html 查看结果

# 扩展应用

这个脚本可以扩展为：

  • 检查图片是否可访问（添加 HTTP 请求验证）
  • 下载所有图片到本地
  • 生成图片资源清单
  • 批量替换图片链接

# 小结

通过正则表达式和简单的文件操作，就能快速实现图片链接的批量提取和预览。这个小工具在处理前端项目资源时非常实用，可以根据实际需求进行定制和扩展。
